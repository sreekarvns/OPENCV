# OPENCV

🧑‍💻 Face Recognition using OpenCV & LFW Dataset

This project implements a face recognition system using OpenCV and the LFW (Labeled Faces in the Wild) dataset.
It focuses on face detection, preprocessing, and matching face pairs for recognition tasks.

🚀 Project Overview

📸 Face detection using OpenCV

🧠 Face recognition using LFW dataset

🔍 Matching and mismatching face pair analysis

📊 Data preprocessing and evaluation

📓 Implemented in Jupyter Notebook
